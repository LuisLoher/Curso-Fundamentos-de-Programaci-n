## 🧠 ¿Qué es programar?

Programar es:

> Dar instrucciones claras y ordenadas a una computadora para resolver un problema.

Una computadora es muy rápida…
pero necesita instrucciones exactas.

---

# 🤖 ¿Qué es un algoritmo?

Un algoritmo es:

> Una serie de pasos ordenados para resolver un problema.

Ejemplo:

1. Tomar cepillo
2. Poner pasta
3. Cepillar
4. Enjuagar

Eso ya es programación sin computadora.

---

# ✍ ¿Qué es el pseudocódigo?

Es una forma estructurada de escribir algoritmos.

No es un lenguaje real.  
Es una herramienta para pensar.

# 1️⃣ ¿Qué es programar?

Programar es:

> Dar instrucciones claras y ordenadas a una computadora para resolver un problema.

Una computadora es muy rápida, pero necesita instrucciones exactas.

---

# 2️⃣ ¿Qué es un algoritmo?

Un algoritmo es:

> Una serie de pasos ordenados para resolver un problema.

Ejemplo cotidiano:

Algoritmo para lavarse los dientes:

1. Tomar cepillo
2. Poner pasta
3. Cepillar
4. Enjuagar

---

# 3️⃣ ¿Qué es el pseudocódigo?

El pseudocódigo es una forma de escribir algoritmos usando lenguaje humano estructurado.

No es un lenguaje real.
Es una herramienta para pensar.
